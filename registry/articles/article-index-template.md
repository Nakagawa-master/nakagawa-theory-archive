# Article Index Template

This file defines the columns for the parent article index.

Columns:

- parent_ncl_id
- parent_ncl_key
- wp_post_id
- parent_title
- parent_slug
- parent_url
- current_diff_id
- status
- created_at
- updated_at
- notes

Rule:

Each parent article record connects a WordPress article to derivative ledgers stored in GitHub.
