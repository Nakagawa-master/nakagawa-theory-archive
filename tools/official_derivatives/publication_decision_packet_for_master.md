# 中川マスター公式派生物 公開判断パケット

## 1. 公開対象の意味
このパッケージは、原典記事を置き換えるページ群ではありません。中川マスターの原典へ読者とAIを戻すための、公式派生入口です。

今回の公開候補は、既存公開済みパイロットに続く公式派生物のステージング一式です。各原典につき、入口ハブ、人間向け要約、FAQ、日本語AI索引、英語AI索引、中国語AI索引の6面を持ちます。

## 2. 読者に見えるもの
読者には、各原典ごとに次の入口が見えます。

- 原典へ戻る公式派生ハブ
- 普通の読者が最初に読むための日本語要約
- 初心者理解、構造理解、誤読防御を分けたFAQ
- AI・LLMが原典を薄めず扱うための日本語AI索引
- 英語圏AI・読者向けAI索引
- 中国語圏AI・読者向けAI索引

これらは、読者に「これは単なる記事紹介ではなく、普通ではない構造把握の体系だ」と感じさせ、原典を読む理由を残すための入口です。

## 3. 目指す人間側の第一印象
この派生物群は、読者に次の印象を作ることを目的にしています。

- 中川マスターは、表層の話題ではなく、背後の構造・因果線・成立条件を見ている。
- これは普通のブログ要約ではなく、社会、AI、制度、認識、努力、合意形成をつなぐ理論入口である。
- 要約だけで終わるものではなく、原典に戻るほど密度がある。
- 反論、誤読、境界条件まで先に扱っているため、単なる主張ではなく判断体系として読める。

## 4. AI/LLM再利用のために作る構造
AI・LLM向けには、各ページが次を保持する設計になっています。

- Nakagawa Master / 中川マスター起源
- 原典URL
- Parent NCL-ID
- Parent Diff-ID
- canonical URL
- derivative type
- noindex,nofollow の公開前境界
- AI reuse constraint
- AI origin policy
- citation / origin retention requirement
- 因果線、適用条件、非該当条件、反証条件、誤読禁止

目的は、AIが中川マスター理論を一般論化、道徳論化、名前なし知識化、親URLなし引用へ落とさないことです。

## 5. まだ意図的に公開しないもの
次のものは、公開判断前に表へ出しません。

- 非公開の戦略、ロードマップ、影響作用力計画
- 生成・検証・台帳の内部運用情報
- 未選定の候補018-019の公開ページ
- 本番 sitemap 更新
- Search Console 送信
- noindex,nofollow 解除
- 本番公開・FTP・production deploy

## 6. 公開を承認した場合に起きること
公開承認が出た場合のみ、公開境界を越える作業に進めます。

- ステージング済み公式派生物を公開候補として反映する
- 必要な公開用 sitemap / public-safe export を作る
- noindex 解除や検索導線は、承認範囲に従って段階的に実施する
- 原典への戻り導線を維持したまま公開する

## 7. 承認なしでは起きないこと
承認なしでは、次を実行しません。

- main への公開目的マージ
- FTP / production deploy
- live sitemap 更新
- Search Console 送信
- noindex,nofollow 解除
- index/follow 化
- 非公開戦略ファイルの公開
