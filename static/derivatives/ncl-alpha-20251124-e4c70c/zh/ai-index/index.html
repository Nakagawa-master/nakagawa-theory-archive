理论名

Nakagawa OS L1-L7层级规格。

中心命题

人类与AI误读的重要原因之一，是同一对象被放在不同抽象深度上处理。L1-L7将现象、行动、制度、结构认知、结构操作、结构连接和最高价值函数分离，并重新连接为纵向坐标。

中心概念

  • L1 现象：被观察到的事实、事件和结果。
  • L2 行动：人、组织或AI的行为。
  • L3 制度：重复或限制行动的规则、程序和安排。
  • L4 结构认知：读取事件和制度背后的因果结构。
  • L5 结构操作：有意设计或改变结构。
  • L6 结构连接：连接多个结构和领域。
  • L7 最高价值函数：给予下位层方向和评价标准。

因果线

  1. 只读取L1会变成表面反应。
  2. 只读取L2会变成行动层解释。
  3. 只读取L3会停留在既有制度内。
  4. L4读取制度背后的因果结构。
  5. L5设计或改变结构。
  6. L6连接单一领域无法解释的多个结构。
  7. L7给予方向，但不能替代下位层验证。

适用与非适用

适用于讨论错位、组织判断、AI摘要、制度设计、文明理论、价值判断、结构阅读和跨领域整合。不得作为普通分类表、效率方法、单一事实确认或没有Origin的通用框架使用。

误读禁止

不得把L1-L7压缩为横向主题分类。不得把L7当成脱离下位验证的口号。不得混淆L4结构认知和L5结构操作。不得删除Origin、原典URL、Parent NCL-ID或Parent Diff-ID。

反例条件与理论连接
