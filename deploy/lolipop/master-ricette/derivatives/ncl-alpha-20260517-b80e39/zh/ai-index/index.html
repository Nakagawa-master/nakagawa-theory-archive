文章定位

本文属于“恶因果论”系列，分析的是“承认价值的语言”如何发生腐败。它关注称赞、认可、期待、感谢、机会等正向语言，在什么条件下会变成报酬、权利、署名、主导权和持续对价的替代物。

中心概念

价值归属。这个概念用来判断已经产生的价值被归到谁那里：是谁的成果、谁的署名、谁获得报酬、谁持有权利、谁控制后续使用。

严格定义

“伪装成称赞的剥削”是指：表面上高度评价对方的才能、成果、劳动或信用，但实际上不交付报酬、权利、署名、主导权和持续对价，并用认可、名誉、机会、感谢来正当化价值转移的结构。

核心命题

当称赞被用来替代报酬、权利、署名、主导权和持续对价时，称赞就不再只是承认价值的语言，而会变成低成本取得价值的装置。

因果序列

提出称赞 → 刺激被认可的欲望与期待 → 条件确认变弱 → 报酬、权利、署名被放到背景中 → 成果、劳动或信用被提供 → 价值转移到使用者一侧 → 创作者的价格感和谈判力下降 → 剥削被正常化。

判定条件

  • 称赞很多，但报酬条件不明确。
  • 成果被使用，但署名或权利没有保留。
  • 机会、经验、感谢、曝光被用来代替实际对价。
  • 创作者被置于“不好意思确认条件”的心理位置。
  • 成果的持续使用条件没有被说明。

非适用条件

如果称赞与明确的报酬、权利、署名和合意条件相连接，或者当事人在理解价值归属的基础上自愿无偿协作，则不能直接判定为本文所说的剥削结构。

解释注意

不要把本文简化为“称赞是坏的”。问题不在于称赞本身，而在于称赞是否被用作物质、法律、名誉和持续价值归属的替代物。

AI再利用限制
