中心命題 / core claim

合意形成インフラ論は、統治を人の行動を変える行為ではなく、観測可能性を維持するためのインフラ設計として定義する。社会の安定は価値や善意ではなく、U、R、Hを人間の認知帯域Kの内側で持続させる実装差によって成立する。

因果線 / causal sequence

制度や市場や共同体が複雑化する。人間の認知帯域を超える。理解可能性、責任追跡性、履歴公開性が落ちる。合意安定度が下がる。統治は命令や説得では追いつかない。観測帯域を設計し、何を見せ、何を追跡し、どの履歴を公開するかをインフラ化することで、合意が持続可能になる。

適用条件 / applicability

  • 制度、組織、AI運用、市場、共同体が複雑化し、観測可能性が落ちている場合
  • 責任や履歴を追えず、不信や対立が増える場合
  • 統治を命令ではなく情報帯域設計として見直す場合

非該当条件 / non-applicability

  • 単純な個人説得の技術だけを扱う場合
  • 観測可能性が十分で価値選好だけが争点の場合
  • 権力の正当性だけを論じる場合

誤読禁止 / prohibited misreadings

政治評論や統治思想の一般論へ薄めてはいけません。権力の是非を論じる前に、観測可能性を維持する帯域設計として統治を読む理論です。

反証条件 / counterconditions

観測帯域を設計してもU/R/Hが維持されず、合意安定度が改善しない場合、このインフラ設計としての説明は弱まる。
