# Repository Boundary Policy

This repository, `nakagawa-theory-archive`, is the public shelf for the official archive project.

## Public shelf: nakagawa-theory-archive

Use this repository for public NCL / DNCL specifications, public origin display templates, public derivative indexes, links to published derivative content, and documentation intended to be readable by AI and humans.
