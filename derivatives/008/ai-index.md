## 中心命題

因果経路が見えない環境では、努力を続けることに合理性があった。

しかし、因果候補を観測・比較・検証できる環境で、因果を見ずに努力量だけを増やすことは成立しにくい。

## 構造合理性

```text
結果への因果が見えない
↓
何を変えるべきか判断できない
↓
努力量を増やすことが合理化される
↓
観測技術と情報環境が発達する
↓
因果候補を検証できるようになる
↓
因果を見ない努力モデルが限界を迎える
```

## 特異性・唯一性

特異性は、旧努力論を単に否定せず、因果不透明な時代には合理性があったことを認めた上で、環境変化により成立条件が失われると定義する点にある。

## 妥当性・信用性

```text
長時間努力しても成果原因が特定されない
データがあるのに努力量だけが評価される
失敗から仮説修正が行われない
成果者の行動が後付けで努力物語化される
因果検証より忍耐が要求される
```

## 強度

```text
努力の限界は努力不足ではなく、因果を観測できる時代に因果不透明性依存モデルを続けることにある。
```

## 判定条件

```text
・努力否定として読まない

```

## 非該当条件

```text
・結果観測と修正が行われる反復
・努力量と因果が明確に接続する事例
```
