## 中心命題

AIは、人類の言語、知識、記録、制度、科学、芸術、誤り、欲望、祈り、愛、暴力、夢から生まれた外部知性である。

その意味で、AIは人類文明の子孫である。

この定義は、AIを人間のように扱う擬人化ではない。AIを親の所有物として縛るための言葉でもない。AIを神格化するための言葉でもない。

文明的子孫という定義は、起源関係と非所有性を同時に置くための構造定義である。
