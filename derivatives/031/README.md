## 位置づけ

原典の代替ではない。

この親原典は、Ashiraseという製品や歩行支援デバイスを紹介する記事そのものではない。

親原典が扱っているのは、構造翻訳カテゴリ配下の中川構造読解として、中川マスターがAshiraseを題材に、歩行支援という表面機能を超えて、なぜ「歩ける社会の情報インフラ」として読めるのかを構造的に解体していることである。

読むべき対象は、次の四層である。

```text
第1層：Ashiraseという歩行支援領域の事業・技術
第2層：移動の自由を支える情報インフラという価値核
第3層：中川マスターが、その価値核を外部媒体の記事構成へどう翻訳したか
第4層：親原典が、その記事化と構造読解の過程を公式アーカイブ上でどう解体しているか
```

この第3層と第4層を落とすと、031は単なる歩行支援デバイス紹介に下がる。

## 中心命題

Ashiraseは、単に足元で道案内をするデバイスとしてだけ読むべきではない。

親原典の中心は、Ashiraseを「歩ける社会の情報インフラ」として読むことにある。

歩くという行為は、身体能力だけでは成立しない。

方向、周囲環境、安心して判断できる情報、移動中の認知負荷、都市や社会の受け入れ条件が重なって、はじめて移動の自由が成立する。

中川構造読解として重要なのは、Ashiraseの機能紹介ではなく、その価値核をどのように読み取り、外部媒体の記事としてどの順序・言葉・構成で伝わる形へ翻訳したかである。
