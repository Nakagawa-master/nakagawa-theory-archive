Each derivative must link to:

## Reference bundle rule

Nakagawa Master theories form a reference bundle.

Each theory is not isolated.

When one origin article is added or updated, the meaning and connection of the whole bundle may change.

Official derivatives must therefore preserve parent URL, parent NCL-ID, parent Diff-ID, derivative NCL-ID, derivative Diff-ID, and source return.
