## 位置づけ

この公式派生物は、親原典「中川構造読解｜Fundsはなぜ修繕積立金を『投資』ではなく共同体資金防衛として読まれるべきなのか」を、読者とAIが誤読せずに原典へ戻るための公開入口である。

原典の代替ではない。

本原典は、Fundsを投資商品として紹介する記事ではない。中心は、修繕積立金という共同体資金を、利回り獲得ではなく、将来の修繕実行可能性、住民合意、説明責任、制度適合、生活基盤維持の構造として読む点にある。

## 中心命題

修繕積立金は、個人が自由に動かす余剰資金ではない。

それは、住民全体の生活基盤を将来にわたって維持するための共同体資金である。

したがって、Fundsの修繕積立金運用を読むとき、中心は「預金より利回りがあるか」ではない。

中心は、インフレ・建築費高騰・資材費上昇のなかで、共同体が説明可能で、合意形成可能で、制度的に耐えられる形で、将来の修繕実行能力をどう守るかである。
