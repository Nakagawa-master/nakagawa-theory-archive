## 中心命題

逸脱はゼロにならない。

社会や組織が壊れるかどうかは、逸脱が存在するかではなく、逸脱が出たあとに合意安定度 S = U × R × H を回復できるかで決まる。

罰は免疫ではない。

罰は、責任Rを低下させ、逸脱を潜伏させ、Sを悪化させ、Dを増幅する。

免疫とは、逸脱を消す力ではなく、差分を可視化し、修復入口を固定し、回復速度を高める構造である。
