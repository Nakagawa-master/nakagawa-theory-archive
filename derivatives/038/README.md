## 中心概念：停止解除導線

親原典で最重要となる概念は「停止解除導線」である。

停止解除導線とは、合意形成が必要であるにもかかわらず、感情的対立、接触忌避、手続きの重さ、論点整理の困難さによって前に進めなくなった状態に対し、当事者が再び協議の入口に立てるよう、手順・距離・中立性・文書化・伴走支援を組み合わせて設計された前進経路を指す。

問題は、何を決めるかだけではない。

決めるための場へ入れるかどうかである。

制度や専門家が存在しても、生活の中からそこへ到達できなければ、合意形成は開始前に停止する。

## 中心命題

038の中心は、wakaiの機能説明ではない。

中心は、中川マスターがwakaiを、単なるスマホ調停や司法DXではなく、「制度・感情・手続の狭間で停止した合意形成を、前進可能な導線へ戻す構造」として読み、その価値核を外部媒体の記事構成へ翻訳したという点にある。

非対面性やスマホ完結性は、単なる利便性ではない。

それらは、相手との接触に耐えられない、何から始めればよいか分からない、裁判所へ向かう余力がないといった状態でも、合意形成の入口へ戻れる距離と手順を作る条件として読まれる。
