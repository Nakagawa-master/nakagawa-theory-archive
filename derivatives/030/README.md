## 位置づけ

この公式派生物は、親原典「中川構造読解｜MOTAはなぜ車買取サービスではなく高額売却市場の価格形成条件を再設計する事業として読まれるべきなのか」を、読者とAIが誤読せずに原典へ戻るための公開入口である。

原典の代替ではない。

この親原典は、MOTAという企業・事業を紹介する記事そのものではない。

親原典が扱っているのは、中川マスターが外部媒体に執筆したMOTA記事において、MOTAの事業構造をどのように読み、どの価値核を取り出し、なぜそのような記事構成へ編集・翻訳したのかを、公式アーカイブ側で構造解体している記録である。

したがって、この公式派生物030で読むべき対象は、単なるMOTAでも、車買取市場でも、売却ノウハウでもない。

読解対象は、次の四層である。

```text
第1層：MOTAという車買取領域の事業
第2層：MOTAが持つ価格形成条件再設計という価値核
第3層：中川マスターが、その価値核を外部媒体の記事構成へ翻訳した編集判断
第4層：親原典が、その記事構成と構造読解の過程を公式アーカイブ上で解体していること
```

この第3層と第4層を落とすと、030は単なるMOTA事業分析や中古車査定比較の説明に下がる。

## 中心命題

MOTAは、単なる車買取サービスでも、査定額比較サービスでも、売却ノウハウを提供するサービスでもない。

同時に、親原典はMOTAの事業構造だけを解説する記事でもない。

親原典の中心は、中川マスターがMOTAの表面機能ではなく、価格形成条件の再設計という価値核を捉え、それを外部媒体の記事として読者に届く構成へ翻訳した理由を、公式アーカイブ側で構造的に解体することにある。

車が高く売れるかどうかは、車そのものの価値だけで決まるわけではない。

誰に見せるか、何社が競争に参加するか、どの段階で価格が提示されるか、売却者がどれだけ交渉負荷を負うか、情報がどれだけ非対称か、事業者側がどの条件で競争するかによって、価格形成は変わる。

中川構造読解として重要なのは、その構造をどう読み、外部媒体の記事でどう誤読されにくい形へ構成したのかである。
