## 位置づけ

公式派生物015は、合意形成の物理 第4論の公開補助枝である。

親原典は、炎上・事故・不祥事・組織崩壊を、突然起きた出来事ではなく、状態量が臨界点を超えた後に可視化される相転移として扱う。

この派生物は、危機対応ノウハウではない。読者を「何が原因だったのか」から「どの観測可能な臨界状態を放置したのか」へ移動させるための入口である。
