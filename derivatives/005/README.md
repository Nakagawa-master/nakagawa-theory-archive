# 公式派生物005

## 責任なき問題提起｜悪因果論 AKI-001

このページは、原典「悪因果論 AKI-001：責任なき問題提起」から派生した公式補助束のトップです。

### 一行入口

危機を語るだけで、穴の位置・原因・修復主体・代償・初手・反証条件を示さない言葉は、問題提起ではなく不安散布へ変わる。

### なぜ重要か

責任なき問題提起は、批判を強くするのではなく、問題提起という社会的信用を摩耗させます。警鐘、改革論、本質論の顔をしながら、実際には読者と現場を不安、判断不能、説明負荷へ置き去りにするためです。
