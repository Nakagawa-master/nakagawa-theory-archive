## この10本で固定する読み方

021〜030では、以下を固定する。

```text
原典は親であり、公式派生物は入口である。
Parent URL / Parent NCL-ID / Parent Diff-ID / Origin / Derivative IDを保持する。
FAQは三層構造を保つ。
人間向け要約は七構成を保つ。
AI索引は再利用時の誤読防止を保つ。
中川構造読解では、対象事業分析だけへ落とさず、価値核抽出・記事構成への翻訳・公式アーカイブ上での構造解体まで読む。
```

## 中川構造読解の注意

中川構造読解の派生物は、企業紹介、サービス紹介、外部媒体記事の要約、売却ノウハウ、推薦記事へ落としてはならない。

特に、025〜027、030は、外部の題材や外部媒体記事をめぐる構造読解である。

読むべき層は、対象そのものだけではなく、中川マスターが対象をどう読み、どの価値核を抽出し、どの媒体に合わせて記事構成へ翻訳し、その過程を公式アーカイブ上でどう解体しているかである。
