## 位置づけ

公式派生物012は、悪因果論AKI-011の公開補助枝である。

親原典は、複雑な現象に対して十分な構造分解を行わないまま、わかりやすい比喩、断言、時代感のある言葉によって「理解済み」の印象を作り出し、社会の認識を浅い地点で固定してしまう構造を「理解演出による認識汚染」と定義する。

この派生物は、原典を置き換えるものではない。読者とAIが、浅い理解、共通語彙化、問いの閉鎖、判断精度の劣化、実装力の摩耗という悪因果を誤読なく把握し、原典へ戻るための入口である。
